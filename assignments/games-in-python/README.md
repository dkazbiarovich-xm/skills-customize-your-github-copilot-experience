
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a classic Hangman game to practice core Python skills, including string manipulation, loops, conditionals, and user input.

## 📝 Tasks

### 🛠️ Word Setup and Hidden Word Display

#### Description
Create the foundation of the game by preparing a word list, selecting one word at random, and displaying hidden progress using underscores.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list.
- Display the current progress using an underscore format such as `_ _ _ _`.

### 🛠️ Player Guesses and Progress Updates

#### Description
Prompt the player to guess letters and update the displayed progress after each guess.

#### Requirements
Completed program should:

- Accept letter guesses from the user.
- Reveal correctly guessed letters in their correct positions.
- Continue showing unrevealed letters as underscores.

### 🛠️ Attempts Tracking and Game Outcome

#### Description
Track incorrect guesses, end the game at the correct time, and communicate the final result to the player.

#### Requirements
Completed program should:

- Track the number of incorrect guesses remaining.
- End the game when the full word is guessed.
- End the game when attempts are exhausted.
- Display clear win and lose messages.
